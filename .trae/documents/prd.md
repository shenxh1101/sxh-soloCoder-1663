## 1. 产品概述

外协订单与质量追溯管理系统是面向中小型制造业（机械加工行业）的全流程管理工具，涵盖外协订单创建、生产跟踪、质量检验、供应商绩效评价等核心业务环节。

- **核心价值**：实现外协订单全生命周期可视化管理，提升质量追溯能力，优化供应商绩效评估
- **目标用户**：技术部、质检部、生产管理部、供应商及企业管理人员
- **解决问题**：外协订单信息分散、质量数据无法追溯、供应商评价缺乏数据支撑

## 2. 核心功能

### 2.1 用户角色

| 角色 | 描述 | 核心权限 |
|------|------|----------|
| 技术部 | 订单创建与图纸管理 | 创建外协订单、上传图纸、选择供应商、查看订单状态 |
| 质检员 | 质量检验与数据录入 | 录入质检结果、记录不良原因、查看质检报告 |
| 管理员 | 系统管理与数据分析 | 供应商管理、绩效评价、数据导出、全流程查看 |

1. **工作台仪表盘**：数据概览、待办事项、关键指标统计

3. **质量管理**：质检录入、不良原因分析、质量报告
4. **供应商管理**：供应商档案、绩效评价、排名榜单
5. **数据导出**：订单明细导出、供应商质量排行榜导出

## 3. 核心流程

### 3.1 外协订单全流程

技术部创建外协订单并上传图纸 → 系统分配给供应商 → 供应商接收并开始生产 → 生产完成后提交质检报告 → 质检员录入检验结果 → 系统自动计算合格率 → 订单完成入库

```mermaid
flowchart LR
    A["技术部创建订单"] --> B["上传图纸"]
    B --> C["选择供应商"]
    C --> D["供应商接单"]
    D --> E["生产加工"]
    E --> F["提交质检报告"]
    F --> G["质检员录入结果"]
    G --> H["系统计算合格率"]
    H --> I["订单入库归档"]
```

### 3.2 供应商绩效评价流程

每月末系统自动汇总 → 统计各供应商订单数据 → 计算合格率与按时交付率 → 生成综合评分 → 输出排名榜单

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：工业蓝（#1e40af）作为主色，体现专业、可靠的制造业特质
- **辅助色**：橙红色（#ea580c）用于警示和强调，绿色（#059669）表示合格/通过
- **中性色**：深灰到浅灰的渐变层次，确保信息清晰可读
- **按钮风格**：圆角矩形按钮，悬浮有阴影和轻微缩放效果
- **字体**：使用现代无衬线字体，标题加粗，正文清晰易读
- **布局风格**：左侧导航栏 + 顶部工具栏 + 主内容区的经典后台管理布局
- **图标风格**：线性图标，统一粗细和圆角

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 工作台 | 数据卡片 | 渐变背景卡片、大数字展示、图标装饰、趋势箭头 |
| 工作台 | 趋势图表 | 平滑折线图、渐变填充、动画效果 |
| 订单列表 | 表格 | 斑马纹行、悬浮高亮、状态标签、操作按钮列 |
| 订单创建 | 表单 | 分组标签、输入框聚焦效果、错误提示、文件拖拽区 |
| 订单详情 | 时间线 | 垂直时间线、节点图标、连接线动画、状态色彩 |
| 供应商绩效 | 排名榜 | 金银铜牌标识、进度条、排名变化箭头 |

### 4.3 响应式设计

- 采用桌面优先设计，主要面向1366px以上分辨率
- 侧边栏可折叠，适配不同屏幕宽度
- 表格支持横向滚动，确保小屏幕下数据完整性
- 表单在窄屏下自动调整为单列布局

### 4.4 交互与动效

- 页面切换时的淡入过渡效果
- 数据卡片悬浮时的微提升动画
- 时间线节点按顺序依次出现的入场动画
- 表单提交成功后的反馈动效
- 进度条和数据加载的骨架屏占位
